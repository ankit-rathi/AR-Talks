# Declaring Foreign Assets in Revised ITR

## Step 1: Download and Install the Offline Utility
1. Browse: https://www.incometax.gov.in/iec/foportal/
2. Navigate to:  
   **Individual / HUF → Salaried Employees → Downloads → Common Offline Utility (ITR 1 to ITR 4)**
3. Download the **Utility ZIP (version 1.2.7, ~35.1 MB)**.
4. Install the utility on your machine.

---

## Step 2: Download Filed ITR JSON
1. Browse: https://eportal.incometax.gov.in/iec/foservices/#/dashboard
2. Navigate to:  
   **e-File → Income Tax Returns → View Filed Returns**
3. For **A.Y. 2025–26**, download the **JSON file** of the filed return.

---

## Step 3: Import JSON and Revise the ITR
1. Import the downloaded JSON into the Offline Utility.
2. Modify the already filed ITR details:
   - Change the **filing section** to **Revised ITR – Section 139(5)**.
   - Go to **Others → Schedule FA → Section A3**  
     (Also fill **Income → Schedule FSI**, if applicable.)
3. In **Schedule FA**, fill in the required foreign asset details  
   (as shared by Kush on WhatsApp).

---

## Step 4: Review and Submit
- Cross-check all other sections carefully.
- Submit and **verify the revised ITR**.
